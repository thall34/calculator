Basic Calculator

Thank you for deciding to check out my basic calculator program. A few things to know beforehand...

  1. This is a basic calculator, so BEDMAS has not been taken into account
  2. The numbers will wrap onto extra lines if you add numbers past the edge of the screen
  3. If you use decimals, the answer will be rounded to the nearest tenth
  4. Dividing by 0 will give an error, to reset use the AC button
  5. If you type something you don't want, hit the ⌫ button and it will erase one character at a time
  6. If you type in two operators in a row, it will replace the old one with the new one
  7. If you type in a second operator after having a full equation, it will calculate and then add the new operator
